### 2. 笔试（半小时）
- 斐波那切数列（黄金分割数列）算法

```java
int fib (int n) {
    int a = 0;
    int b = 1;
    int c;
    for (int i = 2; i <= n; i++) {
        c = a + b;
        a = b;
        b = c;
    }
	return c;
}
```

# 总结
最后，说几点个人看法。

- 平时的积累很重要，要善于积累。
- 开发有一定年限，一定需要懂得深层原理（源码）。
- 持续学习，保持求知欲。
- 不卑不亢，尽最大努力发挥自己了解的，不懂的地方虚心请教。
